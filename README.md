## 2.Описание продукта  

Аналитическая платформа ключевых показателей для Ритейла на данных t2 – является инструментом для аналитики, визуализирующим данные по различным признакам по операторским регионам t2 Российской Федерации. Также реализована тепловая карта гексагонов по количеству трудовых мигрантов, иностранцев и проживающих со слоями точек продаж t2, конкурентов и Миграционными центрами.

---

## 3. Доступ к платформе  

**Ссылка:** [https://geo-platform.tdp.corp.t2.ru/](https://geo-platform.tdp.corp.t2.ru/)  

**Для получения доступа:**  
1. Оформить заявку в BPM/BRO.  
2. В системе «BDO BI Qlik» выбрать услугу «Предоставление доступа к BDO REPORTS».  
3. В разделе логин указать логин от ПК, продукт: **Geo-аналитика**, а также регион/макрорегион доступа. Для доступа ко всем регионам выбрать параметр **«ALL»**.

## 5. Фильтрация данных  

### 5.1 Отчетность  

Раздел «Отчетность»:

![Рисунок 21](https://github.com/user-attachments/assets/f80d219e-370a-40e8-bd0a-179326fc1f96)


Позволяет пользователю настроить:
- Период отчета
- Доступные регионы – регион/регионы, по которому будет отображаться информация. 
Период отчета возможно выбрать из выпадающего списка, по умолчанию выбран последний.  Не поддерживает множественный выбор.

![Рисунок 6](https://github.com/user-attachments/assets/163ecc07-f007-4117-8352-1c0f515b5d2b)


Фильтрация доступна согласно ролевой модели пользователя – от этого варьируются доступные регионы.\
Осуществить фильтрацию по операторским регионам t2 можно тремя способами (при наличии более одного доступного региона), по умолчанию выбраны все (поддерживает множественный выбор):
- Начав вводить название региона в поисковое окно. В выпадающем списке будут отображаться возможные варианты;
- Открыв выпадающий список – выбрать регион;
- Нажав на «Выбрать все».

### 5.2 Слои  

Раздел «Слои»:

![Рисунок 8](https://github.com/user-attachments/assets/eef556d8-2248-433d-829e-1d8bb600abc2)

Позволяет пользователю отобразить на карте информацию об абонентах при помощи гексагонов.<br> 
Возможно настроить слой абонентов из выпадающего списка, по умолчанию выбраны «Проживающие» (не поддерживает множественный выбор):

![Рисунок 9](https://github.com/user-attachments/assets/f3eb9ec2-f46b-4420-924f-d95f76bc3edb)

- **Проживающие** — абоненты с преимущественным нахождением на территории в ночное время;  
- **Высокодоходные** — абоненты с уровнем дохода «Выше среднего» и «Высокий» по методологии Fintech BDO;
- **Иностранцы** — нерезиденты РФ и абоненты иностранных операторов связи;
- **Трудовые мигранты** — нерезиденты РФ, имеющие гражданство стран Ближнего зарубежья (Абхазия, Азербайджан, Армения, Беларусь, Грузия, Кыргыстан, Молдова, Таджикистан, Туркменистан, Узбекистан, Украина, Казахстан).  

Для отображения гексагонов с численной информацией об абонентах, чтобы увидеть результат заданной фильтрации в слое, необходимо включить покрытие.<br>

Гексагоны – это шестиугольники, окрашенные в:
- Серый: 0–499 чел.  
- Голубой: 500–1 499 чел.  
- Желтый: 1 500–7 999 чел.  
- Красный: от 8 000 чел.  

При наведении мышки на какой-то из гексагонов, появляется окно с точным числом:

![Картинке плохо, она лечится.](/image/sample.webp "This is a sample image.")

Также возможно осуществить фильтрацию по гексагонам, которые будут отображаться на карте, для этого необходимо задать интересующее численное значение абонентов:

### 5.3 Точки продаж  

Раздел «Точки продаж»:

![Картинке плохо, она лечится.](/image/sample.webp "This is a sample image.")

Позволяет пользователю платформы настроить точки продаж, которые будут отображаться в кластерах и маркерах (монобрендовая розница, локальные и федеральные партнеры, офисы конкурентов и Миграционные центры).

Представленные точки продаж:

1. **t2** — монобрендовая розница (выбраны по умолчанию);
2. **t2 EX** — монобрендовая розница легкого формата (выбраны по умолчанию);
3. **t2 (open)** — точки в статусе «Временно закрыт» (причина: «Подготовка к новому открытию»).  монобрендовая розница, которые имеют статус «Временно закрыт» с причиной «Подготовка к новому открытию»;
4. **t2 EX (open)** — монобрендовая розница легкого формата, которые имеют статус «Временно закрыт» с причиной «Подготовка к новому открытию»;
5. **t2 FA/FS** — федеральные партнеры;
6. **t2 LA/LS** — локальные партнеры;
7. **MTS** – МТС (выбраны по умолчанию);
8.	**MF** – Мегафон (выбраны по умолчанию);
9.	**BEE** – Билайн (выбраны по умолчанию); 
10.	**other** – Летай, Yota, Мотив;
11. **Миграционные центры** (выбраны по умолчанию).

Осуществить фильтрацию по операторам можно тремя способами (поддерживает множественный выбор):
- Начав вводить название оператора в поисковое окно. В выпадающем списке будет отображаться возможные варианты;
- Открыв выпадающий список – выбрать оператор;
- Нажав на «Выбрать все».

![Картинке плохо, она лечится.](/image/sample.webp "This is a sample image.")

Возможно включить показ «Телеком зоны»: 

![Картинке плохо, она лечится.](/image/sample.webp "This is a sample image.")

Телеком зоны позволяют увидеть потенциальные территории размещения новых точек t2. 

Для каждой точки конкурентов определяется зона в размере 250 метров. При пересечении двух и более точек рассчитывается окружность, которая описывает всю зону – телеком зону. При наличии в данной зоне офиса t2/t2 EX – она будет подсвечиваться серым на карте.

## 6. Поиск точек  

Функционал поиска нужных точек, мест осуществлен двумя способами:
- Поиск по точкам,
- Поиск по координатам.

### 6.1 Поиск по точкам  

Раздел «Отображение точек»:

![Картинке плохо, она лечится.](/image/sample.webp "This is a sample image.")

Позволяет пользователю платформы найти любую точку продаж по трем свойствам:

1.	POS_ID,
2.	Наименование дилера;
3.	Адрес точки.

После нажатия на кнопку «Найти» карта центрируется по заданной точке.

### 6.2 Поиск по координатам  

Раздел «Отображение координат»:

Необходимо ввести долготу и широту интересующей точки и нажать на кнопку «Найти». Далее появится сообщение в нижнем левом углу об успешной фиксации координат:  

![Картинке плохо, она лечится.](/image/sample.webp "This is a sample image.")

## 7. Статистика точек продаж  

Статистика по статусам точек продаж представлена внизу страницы:

![Картинке плохо, она лечится.](/image/sample.webp "This is a sample image.")

Данные изменяются в зависимости от заданных настроек в меню фильтрации. 

Существующие статусы:   
1. **Факт МВ/EX** — точки продаж, которые
	- в статусе «Работает»;
    - в статусе «Временно закрыто» с причиной «Ранее открытая ТТ возобновит работу»;
    - офисы в статусе «Временно закрыто» с причиной «Модернизация».
2. **Работает МВ/EX** — точки продаж, которые имеют статус «Работает»;
3. **Временно закрыто МВ/EX** — точки продаж, которые в статусе «Временно закрыто» с причиной отличной от «Подготовка к новому открытию.
4. **Подготовка к открытию МВ/EX** — точки продаж, которые в статусе «Временно закрыто» с причиной «Подготовка к новому открытию.

МВ – это монобрендовые салоны и модули.

EX – монобрендовая розница легкого формата.

## Функционал компьютерной мыши  

**Управление картой:**

3. **Правая кнопка** — изменение угла отображения карты.
